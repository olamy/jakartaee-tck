Guide to Jakarta Servlet 4.0 TCK Documentation

The Jakarta Servlet Technology Compatibility Kit (TCK) 4.0 documentation includes the following:

  • The Jakarta Servlet TCK 4.0 Release Notes provide the information that you need to install, set up, and run the Jakarta Servlet TCK.

  • The Jakarta Servlet Technology Compatibility Kit 4.0 User's Guide provides the information you need to install, configure, and run the Jakarta Servlet TCK. The User's Guide also contains the rules you need to pass for certification.

  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta Servlet TCK.

  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta Servlet TCK.
